# AI Tutor (Multilingual Offline ML)

## Features
- Intent Classification (Logistic Regression / SVM)
- Lesson Fetcher from CSV
- Quiz Engine
- FastAPI Backend
- Works with Indian languages (Hindi / Marathi)

## How to Run

1. Install dependencies:
    pip install -r requirements.txt

2. Run API:
    uvicorn api.main:app --reload

3. Test in browser:
    http://127.0.0.1:8000/
